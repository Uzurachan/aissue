# 再現手順の詳細

## テスト環境

- 端末: iPhone 13 (実機) / Chrome DevTools モバイルエミュレーション(375x812)
- OS: iOS 17.4
- ブラウザ: Safari 17 / Chrome 126

## 手順

1. `/login` にアクセスする
2. ビューポート幅を375px以下にする
3. メールアドレス・パスワードを入力する
4. ログインボタンをタップする

## 補足

- 幅768px以上(タブレット/PC)では問題なく送信される
- ボタン上に透明なオーバーレイ要素(`div.login-overlay`)が重なっており、タップイベントを奪っている可能性が高い
